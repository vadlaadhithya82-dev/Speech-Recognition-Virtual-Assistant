# Speech-Recognition-Virtual-Assistant
A Python-based voice assistant that uses speech recognition and text-to-speech to perform tasks like playing songs, fetching information, telling jokes, and opening applications.

🎙️ POORI – My Voice Assistant

This is a simple voice assistant I built using Python. It can listen to my voice and perform basic tasks like playing songs, telling time, searching information, and opening applications.

I created this project to learn how speech recognition and text-to-speech work in real-time applications.

🚀 What it can do
* Understand voice commands
* Speak responses
* Play songs on YouTube
* Tell current time
* Search info from Wikipedia
* Tell random jokes
* Open apps like Chrome and VS Code

🛠️ Built With
Python and some useful libraries like:
speech_recognition, pyttsx3, pywhatkit, wikipedia, pyjokes

## ▶️ How to run

1. Install required libraries
2. Run the file:

```
python siri.py
```

Then just speak commands like:
"play song", "what’s the time", "tell me a joke"

## 🗣️ Example Commands

* "Play song name"
* "What’s the time"
* "Who is Elon Musk"
* "Open Chrome"
* "Tell me a joke"

💡 Why I made this

I wanted to build a mini AI assistant to understand how voice-based systems work. This project helped me learn about automation and real-world Python applications.
